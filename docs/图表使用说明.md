# Ant Design Charts 图表使用说明

## 概述

本项目使用 `@ant-design/charts` 作为图表库，版本 2.6.5。该库基于 G2Plot，提供了丰富的图表组件。

## 安装

已在 `package.json` 中安装：
```json
"@ant-design/charts": "^2.6.5"
```

## 常用图表组件

### 1. 条形图 (Bar)
用于横向展示数据，适合车辆数据对比。

```tsx
import { Bar } from '@ant-design/charts';

<Bar
  data={chartData}
  xField="value"
  yField="vehicle_id"
  seriesField="type"
  layout="horizontal"
/>
```

### 2. 柱状图 (Column)
用于纵向展示数据。

```tsx
import { Column } from '@ant-design/charts';

<Column
  data={chartData}
  xField="vehicle_id"
  yField="value"
  seriesField="type"
/>
```

### 3. 双轴图 (DualAxes)
用于展示两个不同量纲的数据，如净收入（柱状）和平均每转收入（折线）。

```tsx
import { DualAxes } from '@ant-design/charts';

<DualAxes
  data={[columnData, lineData]}
  xField="vehicle_id"
  yField={['net_income', 'avg_turn_income']}
  geometryOptions={[
    { geometry: 'column' },
    { geometry: 'line' }
  ]}
/>
```

## 数据格式

图表数据通常为对象数组格式：
```typescript
interface ChartDataItem {
  vehicle_id: string;
  net_income: number;
  avg_turn_income: number;
  revenue?: number;
  turn_count?: number;
}
```

## 配置选项

### 通用配置
- `data`: 图表数据
- `xField`: X轴字段
- `yField`: Y轴字段
- `seriesField`: 分组字段（用于分组/堆叠）
- `height`: 图表高度
- `legend`: 图例配置
- `tooltip`: 提示框配置
- `label`: 标签配置

### 颜色配置
```typescript
color: ({ type }) => {
  if (type === 'net_income') return '#52c41a';
  if (type === 'avg_turn_income') return '#1677ff';
  return '#999';
}
```
